Featured Project

🌾 Kisii AgriPrice - SMS Market Information System

A production-ready web application delivering daily market prices via SMS to small-scale farmers in Kisii, Kenya. Built to address the 20-40% income loss farmers face due to information asymmetry when dealing with middlemen.

Node.js Express JavaScript Africa's Talking API RESTful API Authentication Session Management

Key Features:

  • • Secure admin authentication with bcrypt password hashing
  • • Real-time dashboard with farmer stats, SMS logs, and success metrics
  • • Price management system for 6 local crops with timestamp tracking
  • • Comprehensive farmer CRUD operations (create, read, update, delete)
  • • Bulk SMS delivery system with Africa's Talking API integration
  • • Advanced filtering and search functionality
  • • SMS delivery logging and analytics

Technical Highlights:

  • • Built complete MVP in 3 days under tight deadline
  • • Implemented secure session-based authentication
  • • Scalable architecture ready for MongoDB migration
  • • Responsive design works on all devices
  • • Clean, maintainable codebase with proper documentation
