🌾 Kisii AgriPrice - SMS Market Prices for Farmers

A web application that delivers daily market prices via SMS to small-scale farmers in Kisii, Kenya. Helps farmers avoid exploitation by middlemen by providing real-time local market information.

Node.js Express JavaScript Africa's Talking API RESTful API Authentication

Key Features:

  • Admin authentication & dashboard
  • Price management for 6 crops
  • Farmer registration & management
  • Bulk SMS delivery system
  • Real-time stats & analytics
  • SMS logging & tracking

Impact: 2 registered farmers, 15+ SMS sent successfully, built in 3 days
